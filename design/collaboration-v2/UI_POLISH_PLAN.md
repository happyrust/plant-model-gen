## 6 个改进项

### P1 · 左栏搜索过滤 + 抽屉滚动锁定

**改动**：
1. `Sidebar` 组件加入 `filter` state，对 `envs` 按名称/地区/MQTT 过滤
2. 抽屉（SiteDrawer / ConfigDrawer）打开时 `body.style.overflow='hidden'`，关闭恢复
3. 菜单（ic-menu）打开时点击外部自动关闭

**估时**：10 min

### P2 · 拓扑详情面板增强

**改动**：
1. 右侧详情面板底部增加「诊断历史 · 最近 3 次」折叠区，复用 `MOCK_DIAG_HISTORY` 数据
2. 添加选中节点的关联活跃任务（从 `MOCK_ACTIVE_TASKS` 按 `site_id` 过滤）
3. 节点选中时详情面板添加淡入动画

**估时**：15 min

### P3 · 洞察图表交互增强

**改动**：
1. 堆叠柱状图每根柱子添加 hover tooltip，显示日期、成功数、失败数
2. 图表顶部添加总数标注
3. KPI 卡片添加 hover 微效果（轻微上浮 + 阴影）

**估时**：15 min

### P4 · 活跃任务条 + 站点抽屉关联

**改动**：
1. Pending 任务卡片添加 `waiting` 脉冲边框效果区分 Running
2. 站点详情抽屉新增「进行中任务」section，过滤显示该站点的活跃任务
3. 进度条添加百分比数值标注

**估时**：12 min

### P5 · 空状态 + Tab 切换动画

**改动**：
1. 日志筛选结果为空时显示空状态插图（SVG + 文案）
2. Tab 面板切换添加 `fadeIn` 过渡
3. 拓扑节点选中动画优化（缩放 + 边框变化）

**估时**：12 min

### P6 · 细节打磨

**改动**：
1. 表格行 hover 增加左侧 3px 蓝色边框指示
2. 按钮 hover 添加 `transform: translateY(-1px)` 微动效
3. 搜索结果高亮匹配关键词
4. 响应式断点：当窗口 < 1200px 时左栏折叠

**估时**：15 min
