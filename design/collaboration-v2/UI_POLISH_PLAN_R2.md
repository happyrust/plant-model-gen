# 异地协同 v2 · Hi-Fi 原型 UI 完善计划 · 第二轮

> 制定日期：2026-04-21
>
> 前置：第一轮 P1–P6 已完成（搜索过滤 / 拓扑诊断历史 / 图表 tooltip / Pending 区分 / 空状态 / Tab 动画）
>
> 目标：提升原型交互深度——流向动画、实时数据模拟、操作反馈、响应式、数据可视化增强。

## 5 个改进项

### R2-P1 · 拓扑流向动画粒子 + 节点选中动效

**改动**：
1. SVG 流向箭头上添加沿路径运动的小圆点粒子动画（`<animateMotion>`）
2. `ok` 流向粒子颜色绿色，`warn` 橙色，`bad` 红色 + 暂停动画
3. 节点选中时添加 `scale(1.02)` + 蓝色光晕过渡效果
4. 节点 hover 时显示快速信息 tooltip（站点名 + 状态 + 最近同步时间）

**估时**：15 min

### R2-P2 · 实时模拟引擎

**改动**：
1. `App` 组件添加 `useEffect` 定时器，每 3s 自增 Running 任务进度 (+5~8%)
2. 任务达到 100% 时自动变为 Completed 并弹出 success toast
3. 指挥条「最近同步」时间戳每秒更新
4. ONLINE 指示灯添加定期心跳检测模拟

**估时**：15 min

### R2-P3 · 配置保存反馈 + 操作确认弹窗

**改动**：
1. ConfigDrawer「保存配置」按钮点击后弹出 success toast
2. 「删除协同组」「从协同组删除站点」添加确认弹窗（简易 modal）
3. 「同步」按钮点击后添加到活跃任务并弹 toast

**估时**：12 min

### R2-P4 · 响应式布局

**改动**：
1. 窗口 < 1200px 时左栏折叠为图标条（可切换展开）
2. 拓扑面板 < 900px 时详情面板移至下方（从 grid 2 列变 1 列）
3. KPI 卡片 < 768px 时变 2 列 grid

**估时**：15 min

### R2-P5 · 日志搜索高亮 + 站点概览进度环

**改动**：
1. 日志表格中匹配搜索关键词部分用 `<mark>` 高亮
2. 拓扑详情面板的同步进度用 SVG 环形图展示（已同步 / 总量）
3. 站点表格行内的进度条添加数值标注

**估时**：15 min
