# 异地协同 v2 · Hi-Fi 原型 UI 完善计划 · 第三轮

> 制定日期：2026-04-21
>
> 前置：R1(6 项) + R2(5 项) 已全部完成，原型 1507 行。
>
> 目标：提升原型功能深度——表格交互、数据导出、拓扑全屏、表单、键盘导航。

## 5 个改进项

### R3-P1 · 站点表格列排序 + KPI sparkline

**改动**：
1. 站点表格 thead 各列添加排序按钮（↑↓），点击切换 asc/desc
2. 支持按名称、角色、健康度、状态、最近同步排序
3. KPI 卡片底部添加 7 天迷你 sparkline（SVG polyline），展示趋势

**估时**：15 min

### R3-P2 · 日志行展开/折叠 + CSV 导出

**改动**：
1. 日志表格行点击展开详情区（完整路径、错误堆栈、记录数、大小）
2. 展开区添加「复制错误信息」按钮
3. 「导出 CSV」按钮实现 Blob 下载（包含时间/状态/方向/目标/路径/错误）

**估时**：15 min

### R3-P3 · 拓扑全屏模式 + 节点 hover tooltip

**改动**：
1. 拓扑面板添加「全屏」按钮，点击后面板 fixed 全屏（Escape 退出）
2. 节点 hover 时显示浮动 tooltip（站点名 + 状态 + IP + 最近同步时间）
3. 全屏模式下隐藏右侧详情，改为 tooltip 显示

**估时**：15 min

### R3-P4 · 新增站点表单 + 批量诊断反馈

**改动**：
1. 站点 Tab「+ 新增站点」按钮打开 modal 表单（名称/地区/HTTP/dbnums/备注）
2. 「批量诊断」按钮点击后逐站点执行模拟诊断，结果追加到 toast
3. 站点表格增加 checkbox 列，选中后显示批量操作条

**估时**：18 min

### R3-P5 · 键盘导航 + 无障碍

**改动**：
1. 侧栏协同组列表支持 ↑↓ 键导航、Enter 选中
2. Tab 栏支持 ←→ 键切换
3. 所有可交互元素添加 focus-visible 样式
4. 抽屉/弹窗添加 focus trap
5. ARIA labels 补全

**估时**：12 min
