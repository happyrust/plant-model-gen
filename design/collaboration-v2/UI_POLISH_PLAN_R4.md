## 4 个改进项

### R4-P1 · Toast 自动消失 + 消息队列优化

**改动**：
1. Toast 5 秒后自动消失（带 CSS slideOut 动画）
2. Toast 最多保留 4 条，超出自动移除最早的
3. 鼠标 hover Toast 暂停倒计时
4. Toast 添加进度条指示剩余时间

**估时**：10 min

### R4-P2 · 协同组切换数据联动

**改动**：
1. 切换左栏协同组时，Hero 区域、KPI 卡片、站点列表随之变化
2. 为 3 个协同组各准备差异化 mock 数据（站点数/指标/状态不同）
3. 切换时添加 loading shimmer 过渡（200ms 延迟模拟网络）

**估时**：15 min

### R4-P3 · 拓扑节点拖拽

**改动**：
1. 节点支持鼠标拖拽重定位（SVG transform 更新）
2. 拖拽时流向箭头自动跟随更新
3. 「重置布局」按钮恢复默认位置
4. 拖拽过程中节点添加 drop-shadow 和 scale(1.05) 视觉反馈

**估时**：15 min

### R4-P4 · 健康度仪表盘 + 同步进度环

**改动**：
1. 拓扑详情面板添加 SVG 环形进度图（已同步/总文件）
2. Hero 区域添加迷你健康度指示（4 站点各一个小圆点）
3. 洞察 Tab 的成功率 KPI 用弧形仪表盘替代纯文字

**估时**：12 min
