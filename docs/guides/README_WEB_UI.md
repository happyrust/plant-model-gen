# AIOS 数据库管理平台 Web UI

## 🎯 项目概述

AIOS 数据库管理平台是一个现代化的Web界面，专门用于管理数据库编号7999的数据生成和空间树生成任务。该平台提供了直观的用户界面，支持实时监控、配置管理和任务历史记录。

## ✨ 主要功能

### 🏠 首页
- **欢迎界面**：清晰的功能介绍和快速操作入口
- **功能卡片**：数据生成、空间树生成、配置管理三大核心功能
- **快速操作**：一键创建数据库7999生成任务

### 📊 仪表板
- **系统状态监控**：CPU使用率、内存使用率、数据库连接状态
- **实时图表**：任务执行趋势图、系统资源使用图
- **最近任务**：显示最新的任务执行情况和进度

### 📋 任务管理
- **任务创建**：支持多种任务类型（完整生成、数据生成、空间树生成、网格生成）
- **实时监控**：任务进度条、状态更新、日志查看
- **任务控制**：启动、停止、删除任务
- **快速操作**：预设的数据库7999任务模板

### ⚙️ 配置管理
- **配置模板**：预设的配置模板，支持快速切换
- **参数配置**：数据库编号、生成选项、网格参数等
- **配置预览**：实时显示配置JSON格式
- **配置保存**：持久化配置设置

## 🚀 快速开始

### 1. 启动服务器

#### 真实执行版本（推荐）
```bash
# 编译并启动真实执行版本的Web UI服务器
cargo run --example real_task_execution --features "web_server,ws,gen_model,manifold,project_hd"
```

#### 演示版本
```bash
# 编译并启动演示版本的Web UI服务器
cargo run --example web_server_server --features "web_server,ws,gen_model,manifold,project_hd"
```

### 2. 访问界面

打开浏览器访问：http://localhost:8080

### 3. 执行数据库7999生成任务

#### 方法一：首页快速操作
1. 在首页点击"数据库 7999 生成"卡片中的"立即执行"按钮
2. 系统会自动创建并启动任务
3. 跳转到任务管理页面查看进度

#### 方法二：任务管理页面
1. 访问"任务管理"页面
2. 点击"数据库 7999 生成"快速操作卡片
3. 或者点击"创建新任务"进行自定义配置

## 📱 界面功能详解

### 任务类型说明

| 任务类型 | 说明 | 包含功能 |
|---------|------|----------|
| **完整生成** | 生成所有数据和空间树 | 模型数据 + 网格数据 + 空间树 + 布尔运算 |
| **数据生成** | 仅生成基础数据 | 模型数据 + 网格数据 + 布尔运算 |
| **空间树生成** | 仅生成空间关系 | 房间识别 + 空间树构建 |
| **网格生成** | 仅生成网格数据 | 网格数据 + 布尔运算 |

### 配置参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| **数据库编号** | 要处理的数据库编号列表 | 7999 |
| **生成模型** | 是否生成几何模型数据 | true |
| **生成网格** | 是否生成网格数据 | true |
| **生成空间树** | 是否生成空间树结构 | true |
| **布尔运算** | 是否应用布尔运算优化 | true |
| **网格容差比率** | 网格生成的精度控制 | 3.0 |
| **房间关键字** | 识别房间的关键字模式 | "-RM" |

## 🔧 技术架构

### 前端技术栈
- **HTML5 + CSS3**：现代化响应式界面
- **Tailwind CSS**：快速样式开发
- **Alpine.js**：轻量级JavaScript框架
- **Chart.js**：数据可视化图表
- **Font Awesome**：图标库

### 后端技术栈
- **Rust + Axum**：高性能Web服务器
- **SurrealDB**：现代化数据库
- **Tokio**：异步运行时
- **Serde**：序列化/反序列化
- **Sysinfo**：系统信息监控

### 真实执行引擎
- **gen_all_geos_data**：真实的几何数据生成
- **build_room_relations**：真实的房间关系构建
- **manual_update_aabbs**：真实的AABB树更新
- **update_cal_equip**：真实的设备计算更新
- **update_cal_bran_component**：真实的分支组件计算

### API接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/tasks` | GET | 获取任务列表 |
| `/api/tasks` | POST | 创建新任务 |
| `/api/tasks/:id/start` | POST | 启动任务 |
| `/api/tasks/:id/stop` | POST | 停止任务 |
| `/api/config` | GET/POST | 配置管理 |
| `/api/status` | GET | 系统状态 |

## 🎯 真实执行功能

### 真实任务执行流程

当您在Web UI中创建并启动任务时，系统会执行以下真实的操作：

1. **数据库连接验证**
   - 连接到SurrealDB数据库
   - 验证项目配置和权限
   - 加载必要的SQL函数和脚本

2. **几何数据生成**（如果启用）
   - 调用 `gen_all_geos_data()` 函数
   - 处理指定数据库编号的所有几何体
   - 生成三角网格和执行布尔运算
   - 保存生成的几何数据到数据库

3. **空间索引构建**（如果启用空间树）
   - 加载现有的AABB树或创建新的
   - 调用 `manual_update_aabbs()` 更新空间索引
   - 构建高效的空间查询结构

4. **房间关系构建**
   - 调用 `build_room_relations()` 函数
   - 根据房间关键字匹配房间和面板
   - 计算空间包含关系
   - 保存房间-构件关联关系

5. **设备和组件计算**
   - 调用 `update_cal_equip()` 更新设备计算
   - 调用 `update_cal_bran_component()` 更新分支组件
   - 完成空间关系的最终计算

### 真实系统监控

- **CPU使用率**：通过sysinfo库获取真实的CPU使用情况
- **内存使用率**：监控实际的内存占用情况
- **数据库连接状态**：实时检测SurrealDB连接状态
- **任务进度**：真实反映每个执行步骤的进度
- **错误处理**：捕获并显示真实的执行错误

### 真实数据库信息

- **数据库列表**：从SurrealDB查询真实的数据库编号
- **记录统计**：显示每个数据库的实际记录数量
- **更新时间**：基于会话号推断的最后更新时间
- **可用性状态**：根据记录数量判断数据库可用性
