## 1. 需求理解

### 核心需求

将 `/admin` 站点编排后台补齐为产品可用级别，站点管理功能作为首要开发重点。

### 当前现状（代码验证）

| 体系 | 定位 | 成熟度 |
|------|------|--------|
| `/admin` | 本机多站点进程编排后台 | 核心链路可用，交互缺口明显 |
| `/api/deployment-sites` | 中心注册表站点管理 | 列表/CRUD/导入/健康检查已有 |
| `/console/deployment/sites` | 统一 console 入口 | 仅占位路由，未实装 |

### Step 2：按钮禁用规则 + loading 防重复

#### 2a — 新增 `updateActionButtons()` 函数

```javascript
function updateActionButtons() {
  const site = selectedSite();
  const rt = state.runtime;
  const noSite = !site;
  const status = site?.status || "";
  const parseStatus = site?.parse_status || "";
  const parseRunning = rt?.parse_running || parseStatus === "Running";

  const isStarting = status === "Starting";
  const isRunning = status === "Running";
  const isStopping = status === "Stopping";
  const isBusy = isStarting || isStopping || parseRunning;

  dom.parseBtn.disabled = noSite || isBusy || isRunning;
  dom.startBtn.disabled = noSite || isBusy || isRunning;
  dom.stopBtn.disabled = noSite || isStopping || (!isRunning && !isStarting && !parseRunning);
  dom.deleteBtn.disabled = noSite || isRunning || isStarting || isStopping;
  if (dom.healthcheckBtn) {
    dom.healthcheckBtn.disabled = noSite || !isRunning;
  }
}
```

**禁用矩阵**：

| 状态 | 解析 | 启动 | 停止 | 删除 | 健康检查 |
|------|------|------|------|------|----------|
| 未选中 | ✗ | ✗ | ✗ | ✗ | ✗ |
| Draft/Pending | ✓ | ✓ | ✗ | ✓ | ✗ |
| Parsed | ✓ | ✓ | ✗ | ✓ | ✗ |
| Starting | ✗ | ✗ | ✓ | ✗ | ✗ |
| Running | ✗ | ✗ | ✓ | ✗ | ✓ |
| Stopping | ✗ | ✗ | ✗ | ✗ | ✗ |
| Failed | ✓ | ✓ | 视进程 | ✓ | ✗ |
| parse=Running | ✗ | ✗ | ✓ | ✗ | ✗ |

#### 2b — 修改 `triggerAction()` 增加 loading

```javascript
async function triggerAction(action) {
  const site = selectedSite();
  if (!site) {
    showToast("请先选择站点", "info");
    return;
  }

  const btnMap = { parse: dom.parseBtn, start: dom.startBtn, stop: dom.stopBtn };
  const btn = btnMap[action];
  const labelMap = { parse: "解析", start: "启动", stop: "停止" };
  const label = labelMap[action];

  if (btn) {
    btn.disabled = true;
    btn.dataset.originalText = btn.textContent;
    btn.textContent = `${label}中…`;
  }

  try {
    await request(`/api/admin/sites/${encodeURIComponent(site.site_id)}/${action}`, {
      method: "POST",
    });
    showToast(`${label}请求已提交`, "success");
    await refreshSelectedPanels({ silent: true, skipList: true });
    setupAutoRefresh();
  } catch (error) {
    showToast(error.message, "error");
  } finally {
    if (btn && btn.dataset.originalText) {
      btn.textContent = btn.dataset.originalText;
      delete btn.dataset.originalText;
    }
    updateActionButtons();
  }
}
```

```rust
pub async fn healthcheck_site(site_id: &str) -> anyhow::Result<serde_json::Value> {
    let site = get_site(site_id)?.ok_or_else(|| anyhow::anyhow!("站点不存在: {site_id}"))?;
    let url = format!(
        "http://{}:{}/api/status",
        site.bind_host.as_deref().unwrap_or("127.0.0.1"),
        site.web_port
    );
    let start = std::time::Instant::now();
    let client = reqwest::Client::builder()
        .timeout(std::time::Duration::from_secs(5))
        .build()?;
    match client.get(&url).send().await {
        Ok(resp) => {
            let elapsed_ms = start.elapsed().as_millis();
            Ok(serde_json::json!({
                "healthy": resp.status().is_success(),
                "status_code": resp.status().as_u16(),
                "elapsed_ms": elapsed_ms,
                "url": url,
                "checked_at": chrono::Utc::now().to_rfc3339(),
            }))
        }
        Err(err) => {
            let elapsed_ms = start.elapsed().as_millis();
            Ok(serde_json::json!({
                "healthy": false,
                "error": err.to_string(),
                "elapsed_ms": elapsed_ms,
                "url": url,
                "checked_at": chrono::Utc::now().to_rfc3339(),
            }))
        }
    }
}
```

dom.healthcheckBtn.addEventListener("click", async () => {

  dom.healthcheckBtn.disabled = true;
  dom.healthcheckBtn.textContent = "检查中…";
  try {

      `/api/admin/sites/${encodeURIComponent(site.site_id)}/healthcheck`,

    showToast(msg, result.healthy ? "success" : "error");
  } catch (error) {
    showToast(error.message, "error");
  } finally {
    dom.healthcheckBtn.textContent = "健康检查";
    updateActionButtons();

});

| 决策 | 选择 | 理由 |
|------|------|------|
| 前端框架 | 保持原生 JS | 830 行代码量，引入框架 ROI 不合理 |

| Loading | disabled + textContent 替换 | 零依赖 |
| 日志过滤 | 前端本地 | 仅 120 行/次 |
