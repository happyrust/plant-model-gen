## 性能测试结果

### 测试1：原始嵌套查询（LIMIT 10）

**查询语句：**
```sql
SELECT VALUE [
    id,
    array::last(string::split(NAME, '-')),
    array::flatten(
        (SELECT VALUE
            (SELECT VALUE REFNO FROM PANE WHERE OWNER = $parent.REFNO)
         FROM SBFR WHERE OWNER = $parent.REFNO)
    )
] FROM FRMW
WHERE NAME IS NOT NONE AND '-RM' IN NAME
LIMIT 10;
```

**执行时间：5.211秒**

**返回结果示例：**
```
[FRMW:17496_161440, 'K312', [pe:17496_161442, pe:17496_161461]]
[FRMW:17496_195632, 'R201', [pe:17496_195634, pe:17496_195645]]
```

**性能分析：**
- 平均每条记录：**521ms**
- 查询类型：3层嵌套子查询
- 问题：每个 FRMW 触发多次子查询（N * M * K 复杂度）
