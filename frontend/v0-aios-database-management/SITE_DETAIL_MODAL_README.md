# 部署站点详情弹窗功能

## 功能概述

本次更新为部署站点管理页面添加了弹窗详情功能，用户可以通过弹窗方式查看站点的详细配置信息，而不需要跳转到新页面。

## 主要改进

### 1. 站点卡片增强
- **状态显示**：每个站点卡片现在显示三个关键状态：
  - 数据库工作状态（运行中/启动中/已停止/未知）
  - 数据解析状态（解析完成/解析中/解析失败/未知）
  - 模型生成状态（生成完成/生成中/生成失败/未知）
- **状态图标**：
  - 🟢 绿色✓：完成/运行中
  - 🔵 蓝色时钟：进行中
  - 🔴 红色✗：失败/停止
  - ⚪ 灰色圆圈：未知状态

### 2. 弹窗详情页面
- **运行状态概览**：顶部显示三个状态的实时信息
- **基本信息**：站点名称、负责人、创建/更新时间、描述、访问地址
- **数据库配置**：完整的数据库连接信息和控制功能
- **项目配置**：项目名称、路径、MDB名称、模块信息
- **生成选项**：3D模型生成、网格数据、空间树、布尔运算等选项
- **操作按钮**：编辑配置、删除站点等操作

### 3. 数据库控制功能
- **状态检查**：实时检查数据库运行状态
- **启动/停止**：直接控制数据库的启动和停止
- **状态轮询**：启动后自动检查状态变化

## 技术实现

### 组件结构
```
components/deployment-sites/
├── site-card.tsx              # 站点卡片组件（已更新）
├── site-detail-modal.tsx      # 站点详情弹窗组件（新增）

```

### 关键特性
1. **响应式设计**：弹窗适配不同屏幕尺寸
2. **状态管理**：实时获取和显示各种状态信息
3. **API集成**：与后端API集成获取详细配置
4. **用户体验**：流畅的弹窗动画和交互

## 使用方法

### 查看站点详情
1. 在部署站点管理页面，点击任意站点卡片的"查看详情"按钮
2. 弹窗将显示该站点的完整配置信息
3. 可以查看数据库状态、解析状态、模型生成状态
4. 可以直接在弹窗中控制数据库的启动/停止

### 状态说明
- **数据库状态**：显示数据库服务的运行状态
- **解析状态**：显示数据解析任务的进度
- **模型生成状态**：显示3D模型生成的进度

## API接口

### 获取站点详情
```
GET /api/deployment-sites/{id}
```

### 检查数据库状态
```
GET /api/database/startup/status?ip={ip}&port={port}
```

### 启动数据库
```
POST /api/database/startup/start
Body: { ip, port, user, password, dbFile }
```

### 停止数据库
```
POST /api/database/startup/stop
Body: { ip, port }
```

### 检查解析状态
```
GET /api/parsing/status?siteId={siteId}
```

### 检查模型生成状态
```
GET /api/model-generation/status?siteId={siteId}
```
