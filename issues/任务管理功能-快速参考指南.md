## 🌐 后端API端点速查

### 任务管理API

| 方法 | 端点 | 功能 | 请求参数 | 响应 |
|------|------|------|---------|------|
| GET | `/api/tasks` | 获取任务列表 | `status`, `limit` | `{tasks: Task[], total: number}` |
| POST | `/api/tasks` | 创建任务 | `CreateTaskRequest` | `TaskInfo` |
| GET | `/api/tasks/{id}` | 获取任务详情 | - | `TaskInfo` |
| DELETE | `/api/tasks/{id}` | 删除任务 | - | `{status: "success"}` |
| POST | `/api/tasks/{id}/start` | 启动任务 | - | `TaskInfo` |
| POST | `/api/tasks/{id}/stop` | 停止任务 | - | `TaskInfo` |
| POST | `/api/tasks/{id}/restart` | 重启任务 | - | `TaskInfo` |
| GET | `/api/tasks/{id}/error` | 获取错误详情 | - | `ErrorDetails` |
| GET | `/api/tasks/{id}/logs` | 获取任务日志 | `level`, `limit` | `{task: Task, logs: LogEntry[], total: number}` |
| POST | `/api/tasks/batch` | 批量创建任务 | `BatchTaskConfig` | `{created_tasks: Task[], total: number}` |
| GET | `/api/tasks/next-number` | 获取下一个任务序号 | - | `{next_number: number}` |
| GET | `/api/templates` | 获取任务模板 | - | `TaskTemplate[]` |

## 📊 核心数据结构

### Task (任务)

```typescript
interface Task {
  id: string                          // 任务ID
  name: string                        // 任务名称
  task_type: TaskType                 // 任务类型
  status: TaskStatus                  // 任务状态
  config: DatabaseConfig              // 配置信息
  created_at: number                  // 创建时间 (Unix timestamp ms)
  started_at: number | null           // 开始时间
  completed_at: number | null         // 完成时间
  progress: TaskProgress              // 进度信息
  error: string | null                // 错误信息
  error_details: ErrorDetails | null  // 详细错误
  logs: LogEntry[]                    // 日志列表
  priority: TaskPriority              // 优先级
  dependencies: string[]              // 依赖任务ID
  estimated_duration: number | null   // 预估时长(秒)
  actual_duration: number | null      // 实际时长(毫秒)
}
```

### TaskStatus (任务状态)

```typescript
type TaskStatus = 
  | "Pending"      // 等待中
  | "Running"      // 运行中
  | "Completed"    // 已完成
  | "Failed"       // 失败
  | "Cancelled"    // 已取消
```

### TaskType (任务类型)

```typescript
type TaskType =
  | "DataGeneration"           // 数据生成
  | "SpatialTreeGeneration"    // 空间树生成
  | "FullGeneration"           // 完整生成
  | "MeshGeneration"           // 网格生成
  | "ParsePdmsData"            // 解析PDMS数据
  | "GenerateGeometry"         // 生成几何数据
  | "BuildSpatialIndex"        // 构建空间索引
  | "BatchDatabaseProcess"     // 批量数据库处理
  | "DataParsingWizard"        // 数据解析向导
  | { Custom: string }         // 自定义任务
```

### TaskProgress (任务进度)

```typescript
interface TaskProgress {
  current_step: string                    // 当前步骤描述
  total_steps: number                     // 总步骤数
  current_step_number: number             // 当前步骤编号
  percentage: number                      // 百分比 (0-100)
  processed_items: number                 // 已处理项目数
  total_items: number                     // 总项目数
  estimated_remaining_seconds: number | null  // 预估剩余时间(秒)
}
```

### LogEntry (日志条目)

```typescript
interface LogEntry {
  timestamp: number           // 时间戳 (Unix timestamp ms)
  level: LogLevel            // 日志级别
  message: string            // 日志消息
  error_code: string | null  // 错误代码
  stack_trace: string | null // 堆栈跟踪
}

type LogLevel = "Debug" | "Info" | "Warning" | "Error" | "Critical"
```

### TaskPriority (任务优先级)

```typescript
type TaskPriority = "Low" | "Normal" | "High" | "Critical"
```

## 🔧 常用代码片段

### API调用示例

```typescript
// 获取任务列表
const tasks = await fetchTasks({ status: "running", limit: 50 })

// 启动任务
await startTask(taskId)

// 获取任务日志
const logs = await fetchTaskLogs(taskId, { level: "Error", limit: 100 })

// 创建批量任务
await createBatchTasks({
  template_id: "full_generation",
  batch_config: {
    name_prefix: "数据生成",
    db_nums: [7999, 8000, 8001],
    parallel_execution: false,
    continue_on_failure: true,
  },
})
```

### Hook使用示例

```typescript
// 使用任务列表Hook
const { tasks, loading, error, stats, refetch } = useTasks({
  status: "all",
  autoRefresh: true,
  refreshInterval: 5000,
})

// 使用任务日志Hook
const { logs, task, loading } = useTaskLogs(taskId, {
  autoRefresh: true,
  refreshInterval: 10000,
  limit: 200,
})

// 使用任务操作Hook
const { startTask, stopTask, deleteTask } = useTaskOperations()
```
