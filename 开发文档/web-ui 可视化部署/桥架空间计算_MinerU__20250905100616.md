# 桥架空间计算

# 1. 支架所在桥架号编码

在三维布置设计中，需要实现对电气支吊架所支撑桥架的识别与关联。具体要求为：首先需要获取支吊架所对应的桥架位号信息。由于一个支吊架可能同时支撑多条桥架，因此系统必须能够返回一个桥架位号的集合，而不是单一结果。在三维模型数据中，支吊架的模型类型通常标识为SUPPO，但SUPPO本身并不会直接与桥架发生几何接触，其与桥架的连接依赖于中间的结构单元SCTN（支吊架下的截面单元）。因此，需要先通过SUPPO找到其所有关联的SCTN，再在空间范围内检测这些SCTN与桥架模型的接触关系，最终确定桥架的实体模型并提取其对应的位号。

为保证识别的准确性，系统在实现过程中还需考虑以下几点：

1. 多桥架支撑情况：一个SUPPO可能通过多个SCTN与不同的桥架接触，需要完整输出所有关联桥架的位号集合。

2. 空间接触判定：SCTN与桥架的关系并非通过简单的属性字段体现，而是依赖于空间位置和几何接触判定，需要实现三维碰撞或相邻关系的判断逻辑。

3. 扩展性：除电气支吊架与桥架的关系外，该方法也可以推广到其他类似的支撑-被支撑对象的识别，为后续的设计数据互联互通提供通用的实现思路。

# 2. 预埋板编号

在三维布置设计中，需要通过算法计算支吊架（SUPPO）的生根点位置，并进一步识别其与土建结构（如墙体）的接触关系。具体实现思路为：首先，根据支吊架模型的几何信息确定其生根点坐标，并在空间中找到与其接触的土建墙体位置。随后，需要在该接触位置附近判断是否存在预埋板（FITTING）。

当检测到预埋板时，还需要进一步计算支吊架的接触点是否位于该预埋板的覆盖范围之内。如果支吊架生根点落在预埋板覆盖的有效面积内，则判定该支吊架的生根位置合理、符合设计预期；若不在预埋板范围内，则需标记为异常情况，提示设计人员调整支吊架的布置位置或补充新的预埋板。

为确保判定的精确性，系统在实现过程中需重点考虑以下几个方面：

1. 空间匹配精度：接触点与预埋板的关系需要通过三维空间几何算法精确计算，包括点与面的投影关系、边界范围判定等。

2. 容差设置：由于模型存在绘制误差和数值精度问题，系统应支持一定的空间容差范围判定，避免出现因微小偏差导致的误判。

# 3. 支架定位信息（距墙或定位块）

在支吊架与土建结构的校核过程中，需要通过空间计算来确定支吊架安装所在的墙体位置。由于不同类型的支吊架在结构形式和生根方式上存在差异，因此在计算时应根据支吊架类型选择不同的参照点和计算方法：

# 1.S1型支吊架

对于以圆板为基础的支吊架，其与墙体的接触点并不直接在边缘，而是与圆板中心位置相关。计算时需要先获取圆板的中心点坐标，并从圆板模型中提取Z轴方向的最高点，以此作为参考点来进行墙体距离计算。

随后根据该参考点与墙体的空间关系，找到最近的墙体位置，从而确定支吊架的实际安装面。

# 2.S2型支吊架

对于方钢类的支吊架，其生根点通常位于支撑方钢与墙体的交接位置。在算法实现上，需要首先确定支吊架模型的生根点坐标，并以该点为基准，计算其在X、Y、Z三个方向上与最近墙体的距离。

通过比较不同方向的距离，选取最小值对应的墙体，作为该支吊架的目标安装墙。在整个判定过程中，还需注意以下技术要点：

1. 几何精度：墙体与支吊架之间的空间距离需要采用精确的三维几何算法进行计算，避免因模型误差造成误判。2.墙体搜索范围：在复杂建筑结构中，支吊架周围可能存在多个候选墙体，需要在一定的空间搜索范围内进行筛选，确保能够找到正确的目标墙。3.支吊架类型扩展：除S1、S2型支吊架外，后续还可能存在其他类型的支吊架，需要预留扩展机制，为不同类型支吊架定义对应的参照点和计算逻辑。

# 4. 支架与预埋板相对定位

在完成支吊架与墙体接触位置及预埋板的识别后，还需要进一步计算支吊架生根点与预埋板之间的相对位置关系。具体方法为：首先获取通过空间计算找到的预埋板的中心点坐

标，然后以该中心点作为基准，减去支吊架在墙上的生根点坐标，从而得到一个位置向量。该向量不仅可以反映支吊架生根点与预埋板中心点之间的偏移量，还能够提供偏移的方向信息。

在应用层面，该计算结果具有以下价值：

1. 安装偏差分析：通过比较偏移向量的大小，可以判断支吊架生根点是否落在预埋板的覆盖范围内。如果偏移值小于预埋板半径或有效边界范围，则说明生根点位置合理；反之，则提示存在偏差，需要设计或施工调整。

2. 容差判定：在计算过程中可设置一定的空间容差，当偏移值在允许范围内时仍判定为合理，以避免因模型精度或输入误差造成误判。

# 5.支架与钢结构相对定位

在支吊架与墙体的安装校核过程中，除了需要考虑预埋板的情况，还必须识别另一类常见的结构形式——钢结构。具体实现逻辑为：首先通过空间计算确定支吊架与墙体的接触点。当该接触点未落在预埋板范围内时，系统需要判断其对应位置是否存在钢结构。若存在钢结构，则需要进一步计算支吊架与钢结构之间的相对空间关系。

计算方法是：以钢结构的几何中心点作为基准点，然后减去支吊架对应的参考点坐标，得到两者之间的空间向量。支吊架的参考点因类型不同而有所差异：

S1型支吊架（圆板类）：需先确定圆板的几何中心点，并取其Z轴最高点作为参考点，再与钢结构中心点进行向量差运算。

S2型支吊架（方钢类）：取其生根点作为参考点，与钢结构中心点进行向量差运算。

# 6.托盘跨度（左右）

在支吊架与桥架（Bran）关系的空间校核中，需要首先通过空间计算确定支吊架所支撑的桥架Bran。由于一个支吊架可能同时支撑多条Bran，因此在计算过程中应返回支撑的Bran集合。随后，在每个Bran的维度下，需要进一步寻找与该支吊架相邻的两个支吊架。所谓“相邻”指的是在空间布置上位于同一Bran两侧，且与该支吊架处于同一承载路径的支撑构件。

当找到相邻的两个支吊架后，需要校核三者之间的相对位置关系。具体方法是：分别计算目标支吊架与两侧相邻支吊架之间的中心点距离或生根点距离。通过该距离的计算，可以分析以下几个方面：

# 1.支吊架分布均匀性

如果三个支吊架支撑的是同一个Bran，则相邻支吊架间的距离应保持在合理的设计范围内。

# 2.多Bran支撑情况

若一个支吊架同时支撑多个Bran，则需要分别在每条Bran上独立进行相邻支吊架的距离计算。
