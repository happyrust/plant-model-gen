# 布尔运算架构概述

## 1. 什么是布尔运算

在 3D 模型生成中，布尔运算用于处理几何体之间的"挖孔"操作：

```text
最终模型 = 正实体 - 负实体1 - 负实体2 - ... - 负实体N
```

- **正实体 (Positive)**: 主体几何体，如管道、设备主体
- **负实体 (Negative)**: 需要从主体中减去的几何体，如孔洞、切口

## 2. 两种布尔运算类型

### 2.1 元件库布尔运算

| 属性 | 说明 |
|------|------|
| **函数** | `apply_cata_neg_boolean_manifold` |
| **查询** | `query_cata_neg_boolean_groups` |
| **用途** | 元件库内部几何体的负实体运算 |
| **数据来源** | `has_cata_neg` 标志 + `geo_relate.cata_neg` |
| **结果标记** | `inst_relate.booled = true` |

### 2.2 实例级布尔运算

| 属性 | 说明 |
|------|------|
| **函数** | `apply_insts_boolean_manifold` |
| **查询** | `query_manifold_boolean_operations` |
| **用途** | 设计实例之间的负实体运算 |
| **数据来源** | `neg_relate` / `ngmr_relate` 关系 |
| **结果标记** | `inst_relate.booled_id = {mesh_id}` |


## 4. 模块依赖

```text
gen-model-fork/
├── src/fast_model/
│   ├── mesh_generate.rs        # 入口: booleans_meshes_in_db()
│   ├── manifold_bool.rs        # 核心布尔运算实现
│   ├── pdms_inst.rs            # 负实体关系创建
│   └── loop_model.rs           # LOOP 负实体收集
│
└── aios_core/                   # 依赖库
    ├── query_cata_neg_boolean_groups()      # 元件库布尔查询
    ├── query_manifold_boolean_operations()  # 实例级布尔查询
    ├── ManifoldRust                         # 布尔运算引擎
    └── PlantMesh                            # 网格序列化
```

## 5. 技术栈

| 组件 | 技术 | 说明 |
|------|------|------|
| 布尔运算引擎 | Manifold | 高性能几何布尔运算库 |
| 数据库 | SurrealDB | 图数据库，存储实体关系 |
| 异步运行时 | Tokio | 并发任务处理 |
| 网格格式 | PlantMesh | 自定义二进制格式 |

## 6. 调用链路

```text
gen_all_geos_data()
    │
    ├─▶ gen_full_noun_geos()          # 生成实例数据
    │
    ├─▶ execute_gen_inst_meshes()     # 生成基础 mesh
    │
    └─▶ execute_boolean_meshes()      # 执行布尔运算
            │
            └─▶ booleans_meshes_in_db()
                    │
                    ├─▶ apply_cata_neg_boolean_manifold()
                    │       └─▶ query_cata_neg_boolean_groups()
                    │
                    └─▶ apply_insts_boolean_manifold()
                            └─▶ apply_insts_boolean_manifold_single()
                                    └─▶ query_manifold_boolean_operations()
```
