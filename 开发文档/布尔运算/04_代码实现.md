## 关键函数

### 1. apply_cata_neg_boolean_manifold

元件库布尔运算入口，处理 `has_cata_neg=true` 的实例。

**流程**: 查询 → 分批 → 加载正/负实体 → batch_boolean_subtract → 保存

**重要**: 布尔完成后，原正实体的 `geo_type` 更新为 `CatePos`，`visible=false`

### 2. apply_boolean_for_query

实例级布尔运算（Design 级别），处理有 `neg_relate` 的实例。

**负实体变换计算**:
```rust
// 使用 NegInfo.carrier_world_trans（每个负实体自己的载体世界变换）
let carrier_world_mat = carrier_world_trans
    .as_ref()
    .map(|t| t.0.to_matrix().as_dmat4())
    .unwrap_or(DMat4::IDENTITY);

// 相对变换 = inverse(正实体世界坐标) × 负实体世界坐标
let relative_mat = inverse_pos_world * (carrier_world_mat * geo_local_trans);
```

### 3. load_manifold

### 4. weld_vertices_for_manifold（关键！）

位置: `rs-core/src/geometry/csg.rs`

**作用**: 将 PlantMesh 的非共享顶点模式转换为 Manifold 需要的共享顶点拓扑。

```rust
// 自适应精度计算
let precision = if min_extent < 0.1 { 100000.0 }
    else if min_extent < 1.0 { 10000.0 }
    else if min_extent < 10.0 { 1000.0 }
    else if min_extent < 100.0 { 100.0 }
    else { 10.0 };

// 量化顶点坐标
let quantize = |v: Vec3| -> (i64, i64, i64) {
    ((v.x * precision).round() as i64,
     (v.y * precision).round() as i64,
     (v.z * precision).round() as i64)
};
```

## 导出变换

### OBJ 导出变换逻辑

位置: `src/fast_model/export_model/export_obj.rs`

```rust
// 根据 has_neg 决定变换方式：
// - has_neg=true（booled_id）: local_transform 已经包含世界变换
// - has_neg=false（普通几何体）: 需要 world_transform × local_transform
let combined_transform = if component.has_neg {
    // booled_id: 查询时 transform 已经是 world_trans.d
    instance.local_transform
} else {
    // 普通几何体: inst_transform × geo_transform
    component.world_transform * instance.local_transform
};
```

### 导出逻辑表

| 类型 | has_neg | 变换方式 |
|------|---------|----------|
| **booled_id（布尔结果）** | `true` | `local_transform`（已包含 world_trans） |
| **Pos/Compound（普通几何体）** | `false` | `world_transform × local_transform` |

## 数据库更新

### 实例级布尔

```sql
-- 布尔成功
UPDATE inst_relate SET booled_id = '{mesh_id}', bool_status = 'Success', aabb = aabb:⟨{hash}⟩;

-- 布尔失败
UPDATE inst_relate SET bool_status = 'Failed';
```

### 元件库布尔

```sql
-- 创建布尔结果几何
CREATE inst_geo:⟨{new_id}⟩ SET meshed = true, aabb = aabb:⟨{hash}⟩;

-- 创建新的 geo_relate（布尔结果）
RELATE {inst_info_id}->geo_relate->inst_geo:⟨{new_id}⟩ SET geo_type='Pos', visible=true;

-- 隐藏原正实体
UPDATE {inst_info_id}->geo_relate SET geo_type='CatePos', visible=false 
WHERE geom_refno = pe:⟨{原正实体}⟩ AND geo_type='Pos';
```

## geo_type 类型说明

| geo_type | 含义 |
|----------|------|
| `Pos` | 正实体（可见） |
| `Compound` | 有关联负实体的正实体 |
| `CatePos` | 被布尔替换的原始正实体（隐藏） |
| `DesiPos` | 设计型正实体 |
| `CataNeg` / `CataCrossNeg` | 负实体 |
