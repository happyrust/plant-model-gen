# 布尔运算代码实现

## 核心文件

| 文件 | 作用 |
|------|------|
| `src/fast_model/manifold_bool.rs` | 布尔运算核心实现 |
| `src/fast_model/pdms_inst.rs` | 负实体关系写入 |
| `aios_core` 查询函数 | 数据库查询 |

## 关键函数

### 1. apply_cata_neg_boolean_manifold

元件库布尔运算入口，处理 `has_cata_neg=true` 的实例。

**流程**: 查询 → 分批 → 加载正/负实体 → batch_boolean_subtract → 保存

### 2. apply_insts_boolean_manifold_single

实例级布尔运算，处理有 `neg_relate` 的实例。

**关键**: 坐标变换 `m = wt.inverse() × neg_t × trans`

### 3. load_manifold

加载 mesh 文件并转为 Manifold 对象。

## 数据库更新

```sql
-- 元件库布尔完成
UPDATE inst_relate SET booled = true;

-- 实例级布尔完成
UPDATE inst_relate SET booled_id = '{mesh_id}';

-- 布尔失败
UPDATE inst_relate SET bad_bool = true;
```
