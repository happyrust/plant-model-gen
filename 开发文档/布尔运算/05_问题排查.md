# 布尔运算问题排查

## 1. 常见问题

### 问题 1: 负实体没有被减去

**症状**: 模型没有孔洞

**排查步骤**:

```bash
# 1. 检查负实体关系是否存在
cargo run --bin test_boolean_debug --features gen_model
```

```sql
-- 2. 查询 neg_relate 数量
SELECT count() FROM neg_relate;

-- 3. 查询特定实例的负实体
SELECT * FROM neg_relate WHERE out = pe:{refno};
```

**常见原因**:

- `neg_relate_map` 为空（负实体未收集）
- 配置 `full_noun_enabled_categories` 未包含负实体类别
- 数据源没有负实体数据

### 问题 2: bad_bool = true

**排查 SQL**:

```sql
-- 查询失败的布尔运算
SELECT in, bad_bool FROM inst_relate WHERE bad_bool = true LIMIT 10;
```

**常见原因**:

- 正实体 mesh 加载失败
- 负实体 mesh 加载失败
- 几何体无效（三角形数量为 0）

### 问题 3: booled_id 未更新

**排查 SQL**:

```sql
-- 查询有负实体但未完成布尔的实例
SELECT in FROM inst_relate 
WHERE ((in<-neg_relate)[0] != NONE OR (in<-ngmr_relate)[0] != NONE)
  AND booled_id = NONE
  AND bad_bool != true;
```

## 4. 配置检查

**DbOption.toml**:

```toml
gen_mesh = true
apply_boolean_operation = true
# 确保包含负实体相关类别
full_noun_enabled_categories = []  # 空 = 全部启用
```

## 5. 数据库状态速查

```sql
-- 统计布尔运算状态
SELECT 
  count() as total,
  count(booled = true) as cata_done,
  count(booled_id != NONE) as inst_done,
  count(bad_bool = true) as failed
FROM inst_relate;

-- 负实体关系统计
SELECT count() as neg_count FROM neg_relate;
SELECT count() as ngmr_count FROM ngmr_relate;
```
