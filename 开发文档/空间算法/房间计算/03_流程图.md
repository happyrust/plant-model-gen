# 房间计算流程图

## 1. 整体架构流程

```mermaid
flowchart TB
    Start([开始房间计算]) --> Init[初始化数据库连接<br/>加载配置]
    Init --> BuildMap[构建房间-面板映射<br/>build_room_panels_relate]
    BuildMap --> CheckMap{有房间数据?}

    CheckMap -->|否| End1([返回空结果])
    CheckMap -->|是| Loop[遍历每个房间]

    Loop --> Coarse[粗算阶段<br/>SQLite RTree 查询]
    Coarse --> Fine[精算阶段<br/>射线投射检测]
    Fine --> Save[保存房间关系<br/>room_relate]

    Save --> NextRoom{还有房间?}
    NextRoom -->|是| Loop
    NextRoom -->|否| Stats[生成统计信息]
    Stats --> End2([返回 RoomBuildStats])

    style Start fill:#e1f5e1
    style End1 fill:#ffe1e1
    style End2 fill:#e1f5e1
    style Coarse fill:#fff4e1
    style Fine fill:#e1f0ff
```

## 2. 粗算阶段详细流程

```mermaid
flowchart TB
    Start([粗算开始]) --> LoadPanel[加载面板几何数据<br/>query_insts]
    LoadPanel --> CheckPanel{面板有几何?}

    CheckPanel -->|否| End1([返回空列表])
    CheckPanel -->|是| GetAABB[获取面板 AABB<br/>world_aabb]

    GetAABB --> Query[SQLite RTree 查询<br/>query_overlap]
    Query --> Filter[过滤结果<br/>排除面板本身]

    Filter --> Result([返回候选列表<br/>通常 100-500 个])

    style Start fill:#e1f5e1
    style End1 fill:#ffe1e1
    style Result fill:#e1f5e1
    style Query fill:#fff4e1
```

## 3. 精算阶段详细流程

```mermaid
flowchart TB
    Start([精算开始]) --> LoadMesh[加载面板 TriMesh<br/>load_geometry_with_enhanced_cache]
    LoadMesh --> CheckMesh{加载成功?}

    CheckMesh -->|否| End1([返回空结果])
    CheckMesh -->|是| LoopCand[遍历候选构件]

    LoopCand --> QueryCand[查询构件几何<br/>query_insts]
    QueryCand --> ExtractPts[提取关键点<br/>extract_aabb_key_points]

    ExtractPts --> LoopPts[遍历关键点]
    LoopPts --> RayCast[射线投射检测<br/>is_point_inside_mesh_raycast]

    RayCast --> CountInside[统计内部点数]
    CountInside --> NextPt{还有关键点?}

    NextPt -->|是| LoopPts
    NextPt -->|否| CheckThreshold{>50% 在内部?}

    CheckThreshold -->|是| AddResult[添加到结果集]
    CheckThreshold -->|否| Skip[跳过此构件]

    AddResult --> NextCand{还有候选?}
    Skip --> NextCand

    NextCand -->|是| LoopCand
    NextCand -->|否| End2([返回结果集])

    style Start fill:#e1f5e1
    style End1 fill:#ffe1e1
    style End2 fill:#e1f5e1
    style RayCast fill:#e1f0ff
    style AddResult fill:#ccffcc
```

## 4. 射线投射法原理

```mermaid
flowchart TB
    subgraph "射线投射法判断点是否在内部"
        Point[测试点 P] --> RayZ1[发射 +Z 方向射线]
        Point --> RayZ2[发射 -Z 方向射线]

        RayZ1 --> HitZ1{命中网格?}
        RayZ2 --> HitZ2{命中网格?}

        HitZ1 -->|是| Check1[记录命中]
        HitZ1 -->|否| TryX[尝试 X 方向]
        HitZ2 -->|是| Check2[记录命中]
        HitZ2 -->|否| TryX

        Check1 --> BothHit{两侧都命中?}
        Check2 --> BothHit

        BothHit -->|是| Inside([✅ 点在内部])
        BothHit -->|否| TryX

        TryX --> RayX[发射 ±X 方向射线]
        RayX --> CheckX{两侧命中?}

        CheckX -->|是| Inside
        CheckX -->|否| TryY[尝试 Y 方向]

        TryY --> RayY[发射 ±Y 方向射线]
        RayY --> CheckY{两侧命中?}

        CheckY -->|是| Inside
        CheckY -->|否| Outside([❌ 点在外部])
    end

    style Inside fill:#ccffcc
    style Outside fill:#ffcccc
```

## 5. 几何数据加载流程

```mermaid
flowchart TB
    Start([加载几何数据]) --> CheckCache{缓存中有?}

    CheckCache -->|是| CacheHit[返回缓存数据<br/>记录 cache_hit]
    CheckCache -->|否| TryL0[尝试加载 L0]

    TryL0 --> ExistL0{L0 文件存在?}
    ExistL0 -->|是| LoadL0[加载 L0 mesh]
    ExistL0 -->|否| TryL1[尝试加载 L1]

    TryL1 --> ExistL1{L1 文件存在?}
    ExistL1 -->|是| LoadL1[加载 L1 mesh]
    ExistL1 -->|否| TryL2[尝试 L2/L3...]

    LoadL0 --> BuildTri[构建 TriMesh<br/>ORIENTED + MERGE]
    LoadL1 --> BuildTri
    TryL2 --> BuildTri

    BuildTri --> UpdateCache[更新缓存<br/>记录 cache_miss]
    UpdateCache --> CheckSize{缓存 > 2000?}

    CheckSize -->|是| Cleanup[清理缓存]
    CheckSize -->|否| Return
    Cleanup --> Return([返回 TriMesh])

    CacheHit --> Return

    style Start fill:#e1f5e1
    style Return fill:#e1f5e1
    style CacheHit fill:#ccffcc
    style Cleanup fill:#fff4e1
```

## 6. 房间-面板关系构建

```mermaid
flowchart TB
    Start([构建房间-面板映射]) --> QueryFRMW[查询所有 FRMW 节点<br/>按关键字过滤]

    QueryFRMW --> LoopFRMW[遍历每个 FRMW]
    LoopFRMW --> GetRoomNum[提取房间编号<br/>从 name 或 ESSION]

    GetRoomNum --> QueryPANE[递归查询子节点<br/>收集所有 PANE]

    QueryPANE --> SQL["SELECT array::flatten(<br/>@.{1..2+collect}.children<br/>)[?noun='PANE'].id"]

    SQL --> CheckPANE{有 PANE?}
    CheckPANE -->|否| NextFRMW{还有 FRMW?}
    CheckPANE -->|是| AddMap[添加到映射<br/>(room_num, panel_refno, panels)]

    AddMap --> NextFRMW
    NextFRMW -->|是| LoopFRMW
    NextFRMW -->|否| End([返回映射列表])

    style Start fill:#e1f5e1
    style End fill:#e1f5e1
    style SQL fill:#e1f0ff
```

## 7. 结果保存流程

```mermaid
flowchart TB
    Start([保存房间关系]) --> CheckEmpty{结果为空?}

    CheckEmpty -->|是| End1([返回 OK])
    CheckEmpty -->|否| BuildSQL[构建 SQL 语句列表]

    BuildSQL --> LoopRefno[遍历 within_refnos]
    LoopRefno --> GenRelate["生成 RELATE 语句<br/>pe:panel -> room_relate -> pe:component"]

    GenRelate --> AddSQL[添加到语句列表]
    AddSQL --> NextRefno{还有构件?}

    NextRefno -->|是| LoopRefno
    NextRefno -->|否| Combine[合并所有 SQL]

    Combine --> Execute[批量执行<br/>db.query]
    Execute --> CheckResult{执行成功?}

    CheckResult -->|是| End2([返回 OK])
    CheckResult -->|否| Error([返回错误])

    style Start fill:#e1f5e1
    style End1 fill:#e1f5e1
    style End2 fill:#e1f5e1
    style Error fill:#ffcccc
    style Execute fill:#fff4e1
```

## 8. 两阶段对比

```mermaid
graph TB
    subgraph "粗算 (Coarse)"
        C1[输入: 面板 AABB] --> C2[SQLite RTree]
        C2 --> C3[输出: 候选列表]
        C4[时间: O(log n + k)]
        C5[精度: 低 (仅 AABB)]
    end

    subgraph "精算 (Fine)"
        F1[输入: 候选 + TriMesh] --> F2[射线投射]
        F2 --> F3[输出: 确定列表]
        F4[时间: O(k × 27 × raycast)]
        F5[精度: 高 (几何精确)]
    end

    C3 -->|过滤| F1

    style C2 fill:#fff4e1
    style F2 fill:#e1f0ff
    style C5 fill:#ffcccc
    style F5 fill:#ccffcc
```

## 9. 错误处理流程

```mermaid
flowchart TB
    Start([开始处理]) --> LoadPanel{加载面板成功?}

    LoadPanel -->|否| Warn1[warn: 面板无几何实例]
    LoadPanel -->|是| LoadMesh{加载 TriMesh 成功?}

    Warn1 --> Return1([返回空结果])

    LoadMesh -->|否| Warn2[warn: 加载几何文件失败]
    LoadMesh -->|是| QueryCand{查询候选成功?}

    Warn2 --> Return1

    QueryCand -->|否| Warn3[warn: 查询候选失败]
    QueryCand -->|是| Process[正常处理]

    Warn3 --> Return1

    Process --> Save{保存成功?}
    Save -->|否| Error[error: 保存房间关系失败]
    Save -->|是| Success([返回成功])

    Error --> Return2([返回 0])

    style Start fill:#e1f5e1
    style Success fill:#ccffcc
    style Return1 fill:#fff4e1
    style Return2 fill:#ffcccc
    style Warn1 fill:#ffe1cc
    style Warn2 fill:#ffe1cc
    style Warn3 fill:#ffe1cc
    style Error fill:#ffcccc
```

## 10. 并发处理架构

```mermaid
flowchart TB
    subgraph "主线程"
        Main[build_room_relations] --> Split[分割房间列表<br/>按并发数]
    end

    subgraph "Worker 1"
        W1[tokio::spawn] --> R1[处理房间 1]
        R1 --> R2[处理房间 2]
    end

    subgraph "Worker 2"
        W2[tokio::spawn] --> R3[处理房间 3]
        R3 --> R4[处理房间 4]
    end

    subgraph "Worker N"
        WN[tokio::spawn] --> RN[处理房间 N]
    end

    Split --> W1
    Split --> W2
    Split --> WN

    W1 --> Join[join_all]
    W2 --> Join
    WN --> Join

    Join --> Collect[收集统计信息]
    Collect --> End([返回结果])

    style Main fill:#e1f5e1
    style End fill:#e1f5e1
    style Join fill:#fff4e1
```
