# 模型生成数据库保存总结

> 本文档总结了在模型生成（gen_model）过程中保存到 SurrealDB 的所有数据

## 核心数据表

### 1. 实例数据
- **inst_info**: 实例信息表
- **inst_geo**: 几何体数据表
- **inst_relate**: 实例关系（元素 → 实例信息）

### 2. 几何关系
- **geo_relate**: 几何关系（实例 → 几何体）
- **neg_relate**: 负关系（布尔减法的从属关系）
- **ngmr_relate**: 负GM关系

### 3. 辅助数据（去重存储）
- **trans**: 变换矩阵
- **aabb**: 包围盒数据
- **vec3**: 三维向量（关键点）

### 4. 网格文件
- **assets/meshes/{geo_hash}.mesh**: 三角网格文件

## 保存时机

### 阶段1: 几何体生成
执行函数: `save_instance_data()`
- 保存所有表的数据
- 文件位置: `src/fast_model/pdms_inst.rs:424`

### 阶段2: 网格生成
执行函数: `gen_meshes_in_db()`
- 生成三角网格文件
- 更新 `inst_geo.meshed = true`
- 文件位置: `src/fast_model/occ_generate.rs:41`

### 阶段3: 布尔运算
执行函数: `booleans_meshes_in_db()`
- 处理负实体的布尔减法
- 创建新的 `inst_geo` 记录
- 文件位置: `src/fast_model/occ_generate.rs:84`

## 数据关系

```
pe:{refno} (原始元素)
  └─> inst_relate:{refno}
        ├─> inst_info:⟨id⟩
        │     ├─> geo_relate → inst_geo:⟨geo_hash⟩ → mesh文件
        │     ├─> neg_relate → 其他元素
        │     └─> world_trans → trans:⟨hash⟩
        └─> zone_refno, dt (自动填充)
```

## 数据量参考

| 表名 | 相对元素数量 | 说明 |
|-----|------------|------|
| inst_relate | 1:1 | 每个元素一条 |
| inst_info | 1:1 | 每个元素一条 |
| geo_relate | 1:N | 每个几何体一条 |
| inst_geo | 复用 | 相同几何体共享 |
| trans | 去重后 | 相同变换复用 |
| mesh 文件 | 去重后 | 相同几何体共享 |
